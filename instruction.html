Main settings description:

Parameter Description
license_key Program license key (obtain from developers)
api_id Your Telegram application ID
api_hash Your Telegram application hash
test_mode Test mode (true = test, false = real purchases)
test_address Wallet address for test payments

Account settings:

Parameter Description
name Any name for the account (for convenience)
phone_number Telegram account phone number (with country code)
collection Sticker collection ID for purchase
character Character ID in the collection
currency Currency for purchase ("TON")
count Number of stickers to buy at once
threads Number of threads (recommended 1-3)
max_transactions Maximum transactions (0 = no limit)
seed_phrase TON wallet seed phrase (12-24 words separated by spaces)
snipe_monitor Snipe monitoring settings (optional)

üéØ Operating Modes

‚ö†Ô∏è IMPORTANT: The program has two fundamentally different operating modes depending on the snipe_monitor setting!

üîç Snipe Monitoring Mode (snipe_monitor enabled: true)

How it works:
  • The program waits for NEW collections to appear on the platform
  • When a new collection appears that matches your criteria, the program automatically buys it
  • The collection and character parameters in the config are IGNORED
  • The program works in constant monitoring mode

Example config with snipe monitoring enabled:

{ "name": "Sniper Account", "phone_number": "+1234567890", "collection": 0, // ‚Üê Ignored in this mode "character": 0, // ‚Üê Ignored in this mode "currency": "TON", "count": 1, "threads": 1, "max_transactions": 5, "seed_phrase": "your seed phrase here", "snipe_monitor": { "enabled": true, // ‚Üê SNIPE MODE ENABLED "supply_range": { "min": 100, "max": 10000 }, "price_range": { "min": 500000000, "max": 5000000000 }, "word_filter": ["rare", "limited", "exclusive"] } }

üéØ Direct Purchase Mode (snipe_monitor enabled: false or absent)

How it works:
  • The program immediately buys the specified collection and character from the config
  • No monitoring - direct action
  • The collection and character parameters are REQUIRED
  • Perfect when you know exactly what you want to buy

Example config with direct purchase:

{ "name": "Direct Purchase", "phone_number": "+1234567890", "collection": 123, // ‚Üê REQUIRED collection ID "character": 456, // ‚Üê REQUIRED character ID "currency": "TON", "count": 1, "threads": 1, "max_transactions": 1, "seed_phrase": "your seed phrase here", "snipe_monitor": { "enabled": false // ‚Üê SNIPE MODE DISABLED } }
